# ModParks 機能一覧

## アカウント

- GitHub ログイン
- メール/パスワード認証
- パスワードリセット
- 2FA (TOTP)
- パスキー認証 (WebAuthn)
- プロフィール編集
- API キー管理
- 退会/データエクスポート

## プロジェクト

- 作成（名前/スラッグ/説明/アイコン/タグ/ライセンス/ソースコードURL）
- 公開状態（公開/限定公開/非公開/下書き）
- メンバー管理/権限移譲
- 依存関係（CurseForge/Modrinth の mod も指定可）
- スクリーンショット（featured でカルーセル掲載を選択、最大12枚）

## バージョン

- 登録（バージョン番号/Minecraftバージョン/ローダー/更新内容）
- ダウンロード方式：アップロード/外部URL（GitHub Releases/Modrinth/CurseForge）
- プレリリース/ベータのチャネル区分
- チェンジログの Markdown 対応
- jar メタデータ自動入力（fabric.mod.json/mods.toml/quilt.mod.json/plugin.yml）
- ファイル SHA-256 の算出/表示

## 対応コンテンツ
MOD/プラグイン/リソースパック/データパック/シェーダ/MODパック

## 検索/閲覧

- 名前検索
- 種類/ローダー/タグでのフィルタ
- 並び替え（ダウンロード数/新着/更新）
- カード/リスト表示切り替え
- レシピ表示
- Modrinth/CurseForge からの一括インポート

## コミュニティ

- アイデア投稿/いいね/コメント
- アイデアのステータス管理（募集中/開発中/リリース済み）
- 「このアイデアから生まれた Mod」バッジ
- プロジェクトへのコメント
- ユーザー/プロジェクトのフォロー
- お気に入り
- コレクション/マイリスト（公開/共有）

## 開発者体験 (DX)

- GitHub 連携（OAuth/リポジトリ紐づけ）
- GitHub Release の自動同期（Webhook/Cron）
- CurseForge 作者の所有確認（チャレンジコード方式）
- Modrinth/CurseForge とのダウンロード数同期
- API 提供

## セキュリティ

- jar ヒューリスティックスキャン（同梱実行ファイル/zip slip/remote Class-Path 等）
  - malicious は配布遮断、suspicious は警告表示
  - 異議申請/管理者裁定フロー
- SHA-256 による検証
- レート制限（IP + 識別子ベース）
- セキュリティヘッダ（CSP/HSTS/X-Frame-Options 等）

## 通報/管理

- 通報（著作権侵害/マルウェア疑い/スパム/その他）
- 管理者ダッシュボード（通報/非公開化/ユーザー/アイデア）
- 監査ログ（設定変更/バックアップ/通報処理/非公開化/権限変更/スキャン異議裁定）
- 定期バックアップ（Google Drive 退避対応）

## 通知/連携

- 通知（新規MOD/コメント返信など）
- 更新通知（フォロー済みの新バージョン/Discord Webhook）
- SNS共有（Web Share API/OGP/Twitter Card）

## 共通基盤/その他

- 利用規約/プライバシーポリシー
- 多言語化（日本語/英語）
- レスポンシブデザイン
- ダークモード
- SEO 対策
- PWA 対応
- RSS フィード
- 広告位置プレビュー（config でオフ/プレビュー/オン切替）
- オンボーディングツアー
- 右クリックコンテキストメニュー（開く/コピー/共有など）
