# ModParks 機能一覧

## アカウント

- GitHub ログイン / ログアウト
- メール・パスワード認証（登録・ログイン）
- パスワードリセット
- 2要素認証 (2FA)
- プロフィール編集
- 退会・アカウント削除

## プロジェクト

- 作成（名前 / URLスラッグ / 説明 / アイコン / タグ / ライセンス / ソースコードURL）
- 公開・非公開の状態管理（公開 / 限定公開 / 非公開 / 下書き）
- 情報更新・アイコン変更
- 複数人でのプロジェクト管理（チーム開発）・権限移譲
- 依存関係管理（CurseForge / Modrinth の mod を前提 mod に指定可能）
- スクリーンショットのアップロード（カルーセル掲載を featured で選択、最大12枚・画像のみ）

## バージョン

- 登録（バージョン番号 / Minecraftバージョン / ローダー / 更新内容）
- ダウンロード方式：ModParks へアップロード / 外部URL（GitHub Releases / Modrinth / CurseForge）
- プレリリース / ベータのチャネル区分
- チェンジログの Markdown 対応
- jar メタデータ自動入力（fabric.mod.json / mods.toml / quilt.mod.json / plugin.yml パース）
- ファイル SHA-256 の算出・保存・表示（ユーザーが自分で照合可能）

## 対応コンテンツ

Mod / Plugin / Resource Pack / Data Pack / Shader / Modpack

## 検索・閲覧

- 名前検索
- 作者・タグでのフィルタリング
- タグ機能（複数タグでの分類）
- カテゴリ別表示（便利系 / 建築系 / 冒険系 / 工業系 / 魔術系 / その他）
- ランキング（ダウンロード数順）
- カード / リスト表示の切り替え
- レシピ表示（Mod のアイテム・ブロックのレシピ）
- Modrinth / CurseForge からのプロジェクト一括インポート（下書き取り込み、インポート済みは非表示）

## コミュニティ

- アイデア投稿・いいね・コメント
- アイデアのステータス管理（募集中 → 開発中 → リリース済み）とプロジェクトへの正式リンク
- 「このアイデアから生まれた Mod」バッジ
- コメント・レビュー
- ユーザー / プロジェクトのフォロー
- お気に入り機能
- コレクション / マイリスト作成（公開・共有）

## 開発者体験 (DX)

- GitHub 連携（OAuth ログイン・連携/解除、リポジトリ紐づけ）
- GitHub Release の完全自動同期（Webhook / Cron によるリリース時の自動 Version 作成）
- ダウンロード統計ダッシュボード
- Webhook 提供
- API 提供（外部から情報取得可能）

## セキュリティ

- jar ヒューリスティックスキャン（同梱実行ファイル / zip slip / remote Class-Path 等を検出）
  - malicious 判定はダウンロードを遮断、suspicious は警告表示
  - 誤検知向けに異議申請 → 管理者裁定のフロー
- アップロードファイルの構造検証
- ファイルハッシュ (SHA-256) 検証・可視化
- レート制限（IP + 識別子ベース、認証系エンドポイント保護）
- セキュリティヘッダ（CSP / HSTS / X-Frame-Options 等）

## 通報・管理

- 通報機能（著作権侵害 / マルウェア疑い / スパム / その他）
- 管理者用ダッシュボード（通報確認・プロジェクト非公開・ユーザー管理）
- 監査ログ（設定変更・バックアップ操作・通報処理・非公開化・権限変更・スキャン異議裁定）
- データベースの定期バックアップ（Google Drive への退避対応）

## 通知・連携

- 通知機能（新規 Mod・レビュー返信など）
- Mod の更新通知（フォロー済みプロジェクトの新バージョン / Discord Webhook）
- SNS 共有（Web Share API + リンクコピー、OGP / Twitter Card）

## 共通基盤・その他

- 利用規約 / プライバシーポリシー
- 多言語化（日本語・英語、JSON 管理）
- レスポンシブデザイン（PC / タブレット / スマホ）
- ダークモード
- SEO 対策
- RSS フィード（新着配信）
- 広告位置プレビュー（config でオン/オフ/プレビュー切替、本番はオフ）
- スケルトンローディング / タブのキャッシュ最適化
- オンボーディングダイアログ（アカウントごと1回のみ表示）
