====== ModParks 開発者向けドキュメント Wiki ======

このWikiは、マインクラフト（Minecraft Java Edition）向けMOD・プラグイン等の共有プラットフォーム **ModParks** の仕様や開発者向けドキュメントをまとめているサイトです。

各種API、CLIツール、およびレシピ配信サーバー（ModParks Recipe）の仕様や開発手順を以下から参照できます。

===== ドキュメント目次 =====

==== 1. プラットフォーム基本情報 ====
ModParksの全体設計や、開発環境の構築手順に関するドキュメントです。
  * **[[setup|開発環境セットアップガイド]]** : ローカル開発環境の起動手順、前提条件、環境変数設定など
  * **[[database|データベース設計とスキーマ]]** : Drizzle ORMを用いた主要テーブル定義とリレーション構造

==== 2. API リファレンス ====
ModParks上のデータにアクセス・操作するための RESTful API についてのドキュメントです。
  * **[[api|APIリファレンストップ]]** : APIの基本情報と認証方法について
  * **[[auth|Auth & Users API]]** : ユーザー認証（ログイン）、プロフィール取得、フォロー機能など
  * **[[projects|Projects API]]** : 公開プロジェクト（MODやプラグイン等）の検索、作成、更新機能
  * **[[versions|Versions API]]** : 各プロジェクトに対するリリースファイルの取得およびアップロード
  * **[[comments|Comments API]]** : プロジェクトに対するフィードバックやコメントの管理機能
  * **[[collections|Collections API]]** : ユーザー作成のリスト（コレクション）の管理およびフォロー機能
  * **[[ideas|Ideas API]]** : ユーザーから投稿されたアイデアの取得機能
  * **[[webhooks|Webhook仕様]]** : GitHub Release等と連動する自動取り込みWebhooksの仕様

==== 3. CLI ツールガイド ====
ModParksをコマンドラインから操作するための CLI ツールの仕様です。
  * **[[cli|CLIコマンドガイド]]** : コマンドの使用方法、環境変数、各サブコマンドの一覧

==== 4. レシピ配信 CDN (ModParks Recipe) ====
レシピ画像を動的に生成・配信する CDN/API サーバーについてのドキュメントです。
  * **[[recipe|ModParks Recipe 仕様]]** : レシピ画像の動的配信API・アセット一括登録APIなどの仕様
